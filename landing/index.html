Recipe as the build unit

One recipe declares the system. Steps unlock what it can do.

Heta does not force one RAG pipeline. A vector-only KB, a keyword search KB, a Heta graph KB, or a benchmark runner all use the same composition model. 

# Choose components.
recipe = KnowledgeRecipe(
    parsers=KnowledgeParsers(documents=registry),
    models=KnowledgeModels(
        language=llm,
        embedding=embedding,
    ),
    stores=KnowledgeStores(
        objects=objects,
        vector=vectors,
        sql=sql,
    ),
    steps=(
        ParseDocuments(),
        SplitDocuments(),
        EmbedChunks(),
        IndexVectors(),          # vector_search
        PersistChunks(),         # keyword_search
        *HetaGraphProcedure.build().steps(),
    ),
)
